# Screenshots — Claude debugging scratchpad

Drop any screenshot you want me to look at into this folder.

This folder is **gitignored**, so screenshots stay local — they don't end up
on GitHub or get deployed to Vercel. Filenames don't matter; I read whatever
you drop here.

## Suggested naming so we can find things later
- `meta-dashboard-2026-05-12.png`
- `inbox-mobile-overflow.png`
- `vercel-deploy-error.png`

## How to share
1. Take a screenshot (Mac: `Cmd+Shift+4` to drag a region)
2. Save / drag into this folder
3. Tell me in chat: "I dropped a screenshot in `_docs/screenshots/<filename>`,
   can you look at it?"
4. I'll read it with the file path
